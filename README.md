# Product

Реализован компонент product для отображения продуктовой карточки товара.

## Посмотреть
 Просмотр доступен по [ссылке](https://natgr23.github.io/Product/).

 ![Превью](prev.PNG)

 ## Библиотеки
 React, Vite

 ## Запуск проекта
1. Клонировать репозиторий
   ```sh
   git clone https://github.com/natgr23/product
   ```
2. Установить пакеты
   ```sh
   npm install
   ```
3. Запустить сайт
   ```sh
   npm run dev
   ```
